# Synorix (SNRX)

Modern, premium, dark-themed cryptocurrency website for **Synorix (SNRX)**.

Tagline: **Faster and more usable evolution of Bitcoin**

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Pages

- `/` (Home)
- `/about`
- `/features`
- `/whitepaper`
- `/roadmap`
- `/tokenomics`
- `/community`

## Design

- Dark mode only (deep navy to black gradient)
- Electric cyan accent: `#00f0ff`
- Responsive layout (mobile + desktop)
- Smooth motion and subtle animations

## Run Locally

```bash
npm install
npm run dev
```

Then open: `http://localhost:3000`

## Domain

Planned domain: `synorixcoin.com`
